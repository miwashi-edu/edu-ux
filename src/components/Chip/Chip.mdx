# Chip

A compact component for displaying labels, tags, or selectable items with various visual styles.

## Overview

The Chip component is a versatile element used to display small pieces of information, such as tags, categories, or status indicators. It supports multiple variants, sizes, and can include a remove button for interactive functionality.

## Features

- **Multiple variants**: Default, primary, success, warning, error, and outline styles
- **Three sizes**: Small, medium, and large
- **Interactive states**: Hover effects and focus indicators
- **Removable option**: Optional remove button with callback
- **Accessibility**: ARIA attributes and keyboard navigation
- **Responsive design**: Adapts to different screen sizes

## Usage

```jsx
import Chip from './components/Chip';

// Basic usage
<Chip label="React" variant="primary" />

// Removable chip
<Chip 
  label="TypeScript" 
  variant="success" 
  removable={true}
  onRemove={() => console.log('Removed')}
/>

// Different sizes
<Chip label="Small" size="small" />
<Chip label="Large" size="large" />
```

## Accessibility

- Uses `role="button"` for interactive chips
- Implements proper focus management with `tabIndex`
- Includes `aria-disabled` for disabled state
- Remove button has `aria-label` for screen readers
- Supports keyboard navigation
- High contrast mode support
- Reduced motion support for animations

## Best Practices

1. **Use appropriate variants**: Choose colors that match your design system
2. **Keep labels concise**: Short, descriptive text works best
3. **Consider context**: Use chips in appropriate contexts like tags, filters, or status indicators
4. **Handle removal**: Always provide feedback when chips are removed
5. **Test accessibility**: Ensure chips work with keyboard navigation and screen readers
