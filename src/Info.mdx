# UX Components in Web Development

A categorized reference list of common UX components used in web development, each with a brief description.

## üìÑ Page Layout

### [Header](http://localhost:6006/?path=/docs/page-layout-header--docs)
Topmost section, usually holds logo, nav, and user access. Often includes branding and quick access to key actions. [See Storybook](http://localhost:6006/?path=/docs/page-layout-header--docs)

### [Footer](http://localhost:6006/?path=/docs/page-layout-footer--docs)
Bottom area for copyright, contact, and secondary links. Good for legal info, links, and site-wide actions. [See Storybook](http://localhost:6006/?path=/docs/page-layout-footer--docs)

### [Logo](http://localhost:6006/?path=/docs/page-layout-logo--docs)
Vertical area for nav or tools, left or right aligned. [See Storybook](http://localhost:6006/?path=/docs/page-layout-logo--docs)

### [Main](http://localhost:6006/?path=/docs/page-layout-main--docs)
The primary content area of a page. Should be unique per page for accessibility. [See Storybook](http://localhost:6006/?path=/docs/page-layout-main--docs)

### [Container](http://localhost:6006/?path=/docs/page-layout-container--docs)
A fixed-width wrapper that centers page content. Useful for consistent layouts. [See Storybook](http://localhost:6006/?path=/docs/page-layout-container--docs)

### [Grid](http://localhost:6006/?path=/docs/page-layout-grid--docs)
Structural layout of rows and columns. Enables responsive design. [See Storybook](http://localhost:6006/?path=/docs/page-layout-grid--docs)

### [Section](http://localhost:6006/?path=/docs/page-layout-section--docs)
Thematic division of content. [See Storybook](http://localhost:6006/?path=/docs/page-layout-section--docs)

### [Hero](http://localhost:6006/?path=/docs/page-layout-hero--docs)
Large top area often with image and call-to-action. [See Storybook](http://localhost:6006/?path=/docs/page-layout-hero--docs)

### [Card](http://localhost:6006/?path=/docs/page-layout-card--docs)
Boxed container for grouping related content or actions. [See Storybook](http://localhost:6006/?path=/docs/page-layout-card--docs)
